## 1. 아이폰을 이길 수 없었던 터치감
{{< badge style="info" value="문제인식" >}}

{{% hint %}}
- 동일한 터치 H/W와 펌웨어로 구현했음에도 불구하고 사용감에서 아이폰과 큰 차이를 보임
- 수많은 조직이 존재했으나 터치 인터렉션이 어떤 과정을 통해 동작하는지 전체적 관점에서 다루지 못 했음

## 2. 터치감 정량화 솔루션 연구
{{< badge style="success" value="문제해결" >}}

{{% hint %}}
- Gesture 감성/사용성 Factor 측정 및 최적화를 위한 Solution 개발(Prototype) 돌입
- 스마트폰과 똑같이 생긴 레플리카 장비를 개발 (S/W, H/W 모두 개발)
- 해당 장비를 통해 아이폰 reerse-engineering + 터치 인터렉션 기준 도출
{{% /hint %}}

## 3. 터치감 정량화 솔루션 장비 개발
{{< badge style="success" value="문제해결" >}}

{{% hint %}}
- 연구 성과를 바탕으로 양산 프로세스에 적용될 장비 개발 프로젝트 메이킹
- LG전자 MC사업부에 장비 2기 납품 완료하여 양산 장비로 운영
- 터치 인터렉션 측정 방법 자체를 고안 
- 검출 알고리즘을 만들어서 LABVIEW로 장비 개발
{{% /hint %}}
