# 터치 정확도 구현
[{{< badge style="info" title="최신수정일" value="20251205" >}}]()
{{% hint info %}}
- 조직: LG전자
- 기간: 2009년부터 2010년까지
- 직급: 주임연구원
{{% /hint %}}

## 경쟁사 대비 현저하게 떨어지는 터치인식 성공율

{{< badge style="info" value="문제인식" >}}
{{% hint %}}
- 사용자들의 문제로 치부하기엔 경쟁사는 97%의 터치인식 성공율을 가짐
- 동일한 H/W와 펌웨어로 구현했음에도 불구하고 큰 차이를 보임
- 문제 인식을 위한 터치 좌표 수집 코드 삽입 (당시에는 터치 좌표를 볼 수 없었음)
{{% /hint %}}

{{% hint %}}
- 자사의 QWERTY에 대해 최적 인식영역으로 Simulation 한 결과 96.1%로 개선 가능하며 양산 및 개발 모델 적용 예정(3인치 정전식 전모델)
- Dial에 대해 최적 인식영역으로 Simulation 한 결과, 97.3%로 인식율이 개선되며 양산 및 개발 모델 적용 예정(3인치 정전식 전모델)
- 사용형태에 따라 다른 타점 분포를 보이며 향후 개인특성을 고려한 기준 설정하는 관련 특허 출원
{{% /hint %}}
