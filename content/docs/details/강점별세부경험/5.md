# 이해관계자를 연결하는 문제 해결형 커뮤니케이터
[{{< badge style="info" title="최신수정일" value="20251205" >}}]()

{{% hint info %}}
- 다양한 배경의 이해관계자 사이에서 관점을 조율
- 복잡한 개념을 명확한 언어로 전달
{{%/hint %}}

## 조직에 대한 높은 이해도와 실행력

{{% hint %}}
{{< badge style="info" title="조직 관리 능력" >}}

- 두 개의 스타트업에서 팀 리딩 및 조직 안정화에 핵심 역할 수행
- 세종대학교 산업디자인학과에서 6년간 한국·중국 학생 지도, 교육 조직 운영 경험
- 6년간 직접 사업을 운영하며 대표로서 조직·재무·브랜딩 총괄
- LG전자 생산기술원에서 임직원 대표 역임, 대규모 조직의 문화와 의사결정 구조 이해

{{% /hint %}}

{{% hint %}}
{{< badge style="info" title="조직 개편 능력" >}}
- 조직의 병목·의사소통·권한 구조 문제를 빠르게 파악하고 현실적 대안을 제시
- 몸담았던 모든 스타트업에서 대표에게 ‘조직의 중심축’ 역할로 신뢰 확보
- 불명확한 구조에서 명확한 역할·흐름을 설계해 조직 생산성 개선
{{% /hint %}}

<br>

## 다양한 경험으로 팀내 구심점 역할

{{% hint %}}
{{< badge style="info" title="조직 내 리더쉽" >}}
- 다학제적 백그라운드를 기반으로 주니어에게 지식과 문제해결 프레임을 명확하게 전달
- 대기업·창업·귀촌·건축 등 다양한 삶의 경험에서 나온 균형감으로 팀의 심리적 안정감 형성
- 나이·직급·배경과 무관한 개방적 협업 태도, 주니어에게도 배울 점을 찾는 리더십
- 구성원에 대한 지속적인 감사 표현과 신뢰 형성으로 팀 사기와 결속 강화
{{% /hint %}}

{{% hint %}}
{{< badge style="info" title="경영진 메시지·전략 커뮤니케이션" >}}
- 대표의 구상·아이디어·직감적 방향성을 비즈니스 언어로 정제
- 회사의 정체성과 목표를 투자자/고객이 이해할 수 있는 구조적 메시지로 변환
- 전략·콘셉트·핵심 가치 등을 통합해 회사 외부용 스토리라인 제작
    - 마인드허브 
        - 신규 IR Deck 전체 설계 및 기획 리드
    - 포스페이스랩
        - 신규 IR Deck·회사 소개서·브랜드 톤 재정립
        - 신규 웹사이트 기획/정보구조 설계


{{% /hint %}}
